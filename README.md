# wikiWeb

wikiWeb is a local-first browser application for building and exploring a personal visual web of mathematical knowledge.

This repository currently contains the design specification for the first manual-only prototype.

## Start here

1. Read `docs/prototype-specification.md`.
2. Read `docs/data-model.md`.
3. Read `docs/decision-log.md`.
4. Follow `docs/codex-implementation-brief.md` before writing code.

The prototype should be implemented incrementally. The specification is the source of truth unless explicitly revised.
